# Characters which should be avoided in file names

! exclamation point
# pound
$ dollar sign
% percent
& ampersand
* asterisk
/ forward slash
: colon
? question mark
@ at sign
\ back slash
` backtick
{ left bracket
| pipe
} right bracket
‘ single quotes
“ double quotes
+ plus sign
< left angle bracket
= equal sign
> right angle bracket
  blank spaces
